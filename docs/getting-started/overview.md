# Getting Started

Welcome to the Tailwind Luna Theme documentation! This section will help you get up and running quickly.

## What is Tailwind Luna?

Tailwind Luna is a modern, custom theme for Magento 2.4.7 built with:

- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **Docker** - Containerized environment for consistent development
- **GitHub Actions** - Automated CI/CD workflows
- **MkDocs** - Comprehensive documentation site generator

## Documentation Structure

This documentation provides complete guidance on:

- **Docker Setup** - Container configuration, orchestration, services
- **Installation** - Step-by-step Magento and Composer setup
- **Development** - Theme customization, CSS architecture, extension development
- **Deployment** - Automation, GitHub Actions, best practices

## System Requirements

- Linux/Unix environment (Windows with WSL2)
- Docker & Docker Compose
- Git for version control
- PHP 8.3 (via Docker)
- Python 3.11+ (for MkDocs)

## Key Features

✅ Containerized Magento 2.4.7 with PHP 8.3  
✅ Elasticsearch 7.17.0 integrated  
✅ Tailwind CSS utility framework included  
✅ MkDocs with automated GitHub Pages publishing  
✅ GitHub Actions for automated deployment  
✅ Complete documentation and examples
